# Job Offer Prediction System | Python & Machine Learning

## Overview
Built an end-to-end ML pipeline on 100,000 student records 
to predict whether a student will receive a job offer.

## Process
- Handled 65,000+ missing values using median/mode imputation
- Performed EDA using heatmaps, violin plots, and box plots
- Applied ordinal encoding and standard scaling
- Built and compared two ML models

## Results
- Logistic Regression → 92.86% accuracy
- Random Forest → 100% accuracy

## Key Insights
- LinkedIn was the most used job search platform
- Enterprise companies issued the most offers
- STEM students had the highest representation

## Tools Used
Python | Pandas | NumPy | Scikit-learn | Seaborn | Matplotlib
